Resources

The following resources are intended to be quick references and tools for various topics.

Any tools linked in this section are available on my GitHub page (kr4ekt). This website's repository can also be found here.

  • Survival Bare Necessities - A guide to essential survival skills.
  • First Aid - Basic first aid procedures and tips.
  • Navigation Basics - Tips and tools for effective navigation.
  • Antenna Calculator - A simple tool for calculating antenna lengths.
  • Propagation - A basic propagation calculator.
  • Ham Radio Terms - A glossary of common ham radio terms.
  • Ham Radio Band Plans - A quick reference for band allocations.
  • Logbook Guide - A simple logbook template.
